Project-Bubble-Game
🎯 Bubble Hit Game (Html,Css,JavaScript Project)
📖 Project Description
This is a simple browser-based game built using HTML, CSS, and JavaScript.
The player has to click the correct number (hit number) shown on the screen within a limited time.

🎮 How to Play
A random “Hit Number” is shown at the top
Bubbles with random numbers appear on the screen
Click the bubble that matches the hit number
If correct → Score increases by 10
Game continues until time runs out
⏱️ Game Rules
Total Time: 60 seconds
Correct click = +10 score
Wrong click = no score change
When time ends → Game Over screen appears
🧠 Features
Random bubble generation
Real-time score update
Countdown timer
Dynamic DOM manipulation
Simple game logic using JavaScript
🛠️ Technologies Used
HTML
CSS
JavaScript (DOM manipulation, Events, setInterval)
📂 Project Structure
index.html
style.css
script.js
🚀 How It Works (Logic)
makeBubble() → creates random number bubbles
getNewHit() → selects target number
Click event → checks if clicked number matches target
runTimer() → controls game countdown
